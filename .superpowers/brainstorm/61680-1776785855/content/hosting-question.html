Where do you want to host this?

This affects how the backend (RSVP API) and frontend are deployed.

A

Vercel (recommended)

Free tier, deploys automatically from GitHub, supports serverless API routes. Zero server management.

B

Netlify + Netlify Functions

Similar to Vercel — free tier, GitHub integration, serverless functions for the RSVP API.

C

Self-hosted VPS (e.g. DigitalOcean, Linode)

Full control, runs a Node.js server. Requires more setup and a monthly cost (~$5–12/mo).
